# TOIB (Технологии обеспечения информационной безопасности)
Задание 1
Вариант 5

Описание задачи: Напишите программу на Python, которая считывает файл, содержащий список имен пользователей и паролей, по одной паре в строке (через запятую). Он проверяет каждый пароль на предмет утечки данных в результате утечки данных. Вы можете использовать API «Have I Been Pwned» (https://haveibeenpwned.com/API/v3), чтобы проверить, не была ли утечка пароля. (import requests, hashlib)
